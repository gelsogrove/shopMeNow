How It Works

This test page loads the eChatbot widget script dynamically. You should see a chat icon in the bottom-right corner that you can click to open the chat interface.

The widget will:

  • Generate a unique visitor ID (stored in sessionStorage)
  • Send messages to the backend API
  • Process messages through the LLM pipeline
  • Display responses in real-time
