3. Dados Recolhidos

  • Dados de registo: Nome, email, número de telefone
  • Dados de navegação: Endereço IP, cookies técnicos, sessões
  • Dados de faturação: Informações de pagamento (através de fornecedores externos seguros)
  • Mensagens WhatsApp: Conteúdo das conversas

4. Os Seus Direitos

Ao abrigo do GDPR, tem direito a:

  • Aceder aos seus dados (Art. 15)
  • Retificação de dados incorretos (Art. 16)
  • Apagamento ("direito ao esquecimento", Art. 17)
  • Portabilidade de dados (Art. 20)
