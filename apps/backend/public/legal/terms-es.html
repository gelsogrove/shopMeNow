2. Descripción del Servicio

eChatbot es una plataforma de comercio conversacional basada en IA prestada como servicio gestionado integrado — no es software puro como servicio (SaaS). La Plataforma combina tecnología e integración gestionada, incluyendo:

  • Procesamiento de chatbot con IA para conversaciones automatizadas con clientes por WhatsApp, mediante Grandes Modelos de Lenguaje (LLM) a través de OpenRouter (con enrutamiento a Anthropic Claude y otros modelos);
  • Soporte multilingüe — la IA interpreta y traduce dinámicamente las conversaciones;
  • Integración con la API de WhatsApp Business, gestionada a través de la API oficial de Meta o WasenderAPI;
  • Backend de e-commerce multitenant — catálogo de productos, pedidos y gestión de citas;
  • Analítica y panel de operador (backoffice) que permite monitorización en tiempo real e intervención manual en las conversaciones;
  • Procesamiento de pagos para suscripciones de Operadores a través de PayPal. eChatbot no almacena datos de tarjetas de pago.

eChatbot puede añadir, modificar o eliminar funcionalidades en cualquier momento, con un aviso previo razonable para cambios materiales.
