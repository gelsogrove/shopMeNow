2. Descrição do Serviço

A eChatbot é uma plataforma de comércio conversacional baseada em IA prestada como serviço gerido integrado — não é software puro como serviço (SaaS). A Plataforma combina tecnologia e integração gerida, incluindo:

  • Processamento de chatbot com IA para conversas automatizadas com clientes pelo WhatsApp, mediante Grandes Modelos de Linguagem (LLM) através do OpenRouter (com encaminhamento para Anthropic Claude e outros modelos);
  • Suporte multilingue — a IA interpreta e traduz dinamicamente as conversas;
  • Integração com a API do WhatsApp Business, gerida através da API oficial da Meta ou WasenderAPI;
  • Backend de e-commerce multi-inquilino — catálogo de produtos, encomendas e gestão de marcações;
  • Análise e painel do operador (backoffice) que permite monitorização em tempo real e intervenção manual nas conversas;
  • Processamento de pagamentos para subscrições de Operadores através do PayPal. A eChatbot não armazena dados de cartões de pagamento.

A eChatbot pode adicionar, modificar ou descontinuar funcionalidades a qualquer momento, com aviso prévio razoável para alterações materiais.
