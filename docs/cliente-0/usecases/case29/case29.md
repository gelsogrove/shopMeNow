# Case 29 — El cliente menciona cámaras o revisión técnica

## SCENARIO

**Scenario 29.1 — Escalation:** El cliente menciona cámaras de vigilancia, AJAX, o soporte técnico. El chatbot reconoce que requiere investigación externa y escala sin hacer promesas de comprobación inmediata.

## ACCEPTANCE CRITERIA

- El bot reconoce mención de cámaras/AJAX.
- El bot NO promete acceso a cámaras desde el bot.
- El bot escala para investigación manual.
