# Case 27 — El cliente pide una compensación concreta

## SCENARIO

**Scenario 27.1 — Escalation:** El cliente exige compensación específica (secadora gratis, código nuevo, etc.). El chatbot no promete la compensación, sino que escala para que un operador la evalúe.

## ACCEPTANCE CRITERIA

- El bot reconoce la solicitud de compensación.
- El bot NO promete ni confirma la compensación desde el bot.
- El bot escala sin hacer promesas.

## CONVERSATION — Case 27 — Scenario 27.1

**Usuario:** Quiero una secadora gratis por las molestias.
**Bot:** Vamos a revisar tu caso para ayudarte con la solución más adecuada. ¿En qué lavandería estás?

**Usuario:** Goya.
**Bot:** Gracias. ¿Qué número de secadora?

**Usuario:** La 8.
**Bot:** De acuerdo. Vamos a pasar tu caso a revisión para que se valore la compensación más adecuada. ¿Cómo te llamas?

**Usuario:** David.
**Bot:** Gracias David. Un operador revisará tu solicitud y se contactará contigo a la mayor brevedad posible para explicarte la solución. ¿Aceptas recibir una llamada telefónica?
