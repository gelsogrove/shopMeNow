# Aviso de Privacidade e GDPR - Português

## 1. Controlador de Dados

A gestão dos seus dados pessoais é realizada pelo controlador de dados indicado no registro do serviço ShopME.

## 2. Finalidade do Processamento

Os seus dados pessoais são processados para os seguintes fins:
- Fornecer serviços de comunicação via WhatsApp
- Gerenciar pedidos e transações comerciais
- Enviar notificações e atualizações de serviço
- Cumprir com obrigações legais e fiscais
- Melhorar a experiência do usuário

## 3. Base Legal

O processamento dos seus dados é baseado em:
- Execução do contrato (Art. 6, par. 1, lit. b RGPD)
- Interesse legítimo (Art. 6, par. 1, lit. f RGPD)
- Cumprimento de obrigações legais (Art. 6, par. 1, lit. c RGPD)

## 4. Categorias de Dados Processados

- Dados de identificação (nome, sobrenome, número de telefone, email)
- Dados de comunicação (mensagens do WhatsApp, histórico de chat)
- Dados comerciais (pedidos, preferências de produtos)
- Dados de localização (país de origem do número de telefone)
- Dados técnicos (IP, tipo de dispositivo, cookies)

## 5. Destinatários dos Dados

Os seus dados podem ser compartilhados com:
- Fornecedores de serviços do WhatsApp (Meta Platforms)
- Provedores de hospedagem e infraestrutura em nuvem
- Parceiros comerciais e prestadores de serviços
- Autoridades públicas quando exigido por lei

## 6. Período de Retenção de Dados

Os dados são retidos por:
- Toda a duração do relacionamento comercial
- 10 anos por obrigações fiscais e contábeis
- Até 90 dias após a exclusão da conta por motivos de segurança

## 7. Seus Direitos

Você tem o direito de:
- **Acesso** - Acessar seus dados pessoais
- **Retificação** - Corrigir dados imprecisos
- **Apagamento** - Deletar seus dados (direito ao esquecimento)
- **Limitação** - Limitar o processamento
- **Portabilidade** - Receber seus dados em formato estruturado
- **Objeção** - Objeter ao processamento
- **Reclamação** - Apresentar uma reclamação à autoridade supervisora

## 8. Como Exercer Seus Direitos

Para exercer os direitos mencionados acima, entre em contato com o controlador de dados através de:
- Email: privacy@empresa.com
- Formulário de contato no serviço
- Endereço postal fornecido pelo controlador

## 9. Segurança de Dados

Implementamos medidas de segurança técnicas e organizacionais para proteger seus dados contra acesso não autorizado, alteração, perda ou destruição.

## 10. Transferências Internacionais

Alguns dados podem ser transferidos para países terceiros com garantias adequadas de proteção de acordo com a regulação do RGPD.

## 11. Alterações a Este Aviso

Nos reservamos o direito de atualizar este aviso. Notificaremos você sobre alterações materiais.

## 12. Informações de Contato

Para qualquer dúvida sobre o processamento dos seus dados:
- **Encarregado de Proteção de Dados**: dpo@empresa.com
- **Autoridade Supervisora**: Autoridade de Proteção de Dados do seu país
